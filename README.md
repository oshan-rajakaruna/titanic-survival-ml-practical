# Titanic Survival Machine Learning Practical

This project is an end-to-end machine learning practical using the Titanic survival dataset.

## Project Goal

The goal of this project is to predict whether a passenger survived or not based on passenger information such as age, gender, passenger class, fare, and embarkation point.

## Problem Type

This is a binary classification problem.

Target column:

- `Survived`
- `0` = Not Survived
- `1` = Survived

## Workflow

1. Load dataset
2. Check shape, head, and info
3. Identify target column
4. Check missing values
5. Check duplicates
6. Clean data
7. Perform EDA
8. Split X and y
9. Encode categorical values
10. Train/test split
11. Scale data
12. Train model
13. Predict
14. Evaluate
15. Explain results
